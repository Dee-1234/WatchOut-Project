WatchOut is a real-time, collaborative Pomodoro timer application built to boost productivity. By combining a classic time-management technique with real-time WebSocket communication, users can manage their focus sessions seamlessly.

# Features
1. Pomodoro Timer: Standard focus and break intervals to maximize productivity using the Pomodoro technique.
2. Real-Time Synchronization: Powered by WebSockets to ensure instant state updates across connected clients.
3. Intuitive UI: Responsive and modern user interface for a smooth user experience.
4. Robust Backend: Built with a scalable architecture to handle persistent real-time connections.

# Tech Stack
# Backend
1. Java / Spring Boot: Core application framework.
2. Spring WebSockets / STOMP: For real-time, bi-directional client-server communication.
3. Maven: Dependency management and build automation.
